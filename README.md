# My First Portfolio

A collection of HTML projects demonstrating basic web development skills.

## Structure

- `html_basic/` - Basic HTML exercises
  - `base_index.html` - Simple HTML structure
  - `index.html` - Enhanced webpage with content and navigation
  - `tweets.html` - Tweet embedding page
- `portfolio/` - Personal portfolio website
  - `index.html` - Complete portfolio with projects and contact info

## Features

- Valid HTML5 markup
- Semantic structure
- Responsive navigation
- Professional portfolio layout

## Usage

Open any HTML file in a web browser to view the pages.
